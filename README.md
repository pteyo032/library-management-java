# Library Management System (Java CLI)

A clean, beginner-friendly Java project demonstrating object-oriented programming, collections, and exception handling.

## Features
- Add / list / search / remove books
- Borrow / return books (availability tracking)
- Simple CLI menu (terminal)

## Tech
- Java 17+ (works with Java 11+ in most cases)
- Maven (build/test/run)
- JUnit 5 tests

## Run
```bash
mvn -q test
mvn -q exec:java
```

## Notes (Interview-ready)
- Uses **OOP** (Book, Library) and custom exceptions
- Uses **Collections** (HashMap) for fast lookup
- Clear separation: domain models vs CLI

Fluent Java CLI library manager using OOP, collections, exceptions, and unit tests (JUnit).
